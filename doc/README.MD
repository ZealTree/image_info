#### 먫썛혥혪혜햫햣햫햦혪 햨 혝햟햠햩햦혡햣

- **游릭 (행햣햩햣햫햟혪 햡햟햩쮐햨햟)**: 햆햟햫햫혦햣 햡햟햟햫혝햦쮏쒫썛쫧쫧 햢쮐혝혞햫혦 햢햩혪 쒬햣혠 혟햟햧햩쮏 햢햟햫햫쮏 혟쮐햪햟혝햟.
- **游댮 (햌햟혜햫혦햧 햨햣혜혝햦햨)**: 햆햟햫햫혦햣 햫햣햢쮐혝혞햫혦 햢햩혪 햢햟햫햫쮏 혟쮐햪햟혝햟.
- **游리 (했햣햩혝혦햧 햨혞햤쮏)**: 햆쮐혝혞햫쮐혝혧 햢햟햫햫혦혠 향햟쒫쟴햦혝 쮐 햨쮏쫧쥄햣혝햫쮏 혟햟햧햩햟 (햫햟햦햪햣, 햫햟햩햦혢햦혪 EXIF, DPI 햦햩햦 혡쒫왐쮏쒫쮏 쮐혝햟햫혜혝쒫).
- **游댮* (GIF  QImage)**: 햆햩혪 GIF-혟햟햧햩쮏 `QImage` 햫햣 햦혜쮏혧향혞햣혝혜혪 햫햟혪햪혞혩, 혝햟햨 햨햟햨 쮏쫧 쮏쐄햟햠햟혝혦쒫썜뀀혜혪 혢햣햣향 `QMovie`. 먫썛햪햣혦 햦 햡햩혞햠햦햫햟 혡쒫왐햟 햢쮐혝혞햫혦 혢햣햣향 `QImage`  햪햣혝쮏얧 `display_gif_info`, 햫 혝쮏혧햨 햨햟햨 햢쮏쮏햫햦혝햣햩혧햫햟혪 쮏쒫왐햨햟.
- **游리** (HEIC)**: 쮏얧얧왐햤햨햟 HEIC 향햟쒫쟴햦혝 쮐 쒫왐혜햦햦 Pillow 햦 햫햟햩햦혢햦혪 햠햦햠햩햦쮐햣햨햦 `libheif`. 햄햣향 혜쮏쮐쒫왐혜혝쒬혩혤햣햧 쮏얧얧왐햤햨햦 햦향쒫햣혢햣햫햦햣 햪햣혝햟햢햟햫햫혦혠 혢햣햣향 PIL 햪쮏웷왐 향햟쒫왐혣햦혝혧혜혪 쮐걣쟳쐃쥃쮏.
- **WEBP/TIFF**: EXIF 햦 GPS-햢햟햫햫혦햣 햪쮏혞혝 햠혦혝혧 햢쮐혝혞햫혦  햫햣햨쮐쮐혦혠 혟햟햧햩햟혠, 햫 혨혝 햫햣 혪쒫혪햣혝혜혪 혜혝햟햫햢햟혝쮏 (햫햟햦햪햣, WEBP 햣햢햨 혜쮏얧왐햤햦혝 EXIF).
- **PNG-햪햣혝햟햢햟햫햫혦햣**: 햎쮏혞혝 쒫쥃혩혢햟혝혧 혝햟햨햦햣 쮏혪, 햨햟햨 `gamma`, `icc_profile`, 햫 햦혠 햫햟햩햦혢햦햣 햫햣 햡햟햟햫혝햦쮏쒫썛쫧.
- **GPS-햢햟햫햫혦햣**: 햞쒫혪혩혝혜혪 쮏얧쨿쫧쮏웷왐혝쒫쮏 EXIF 햦 향햟쒫쟴혪혝 쮐 혝쮏, 혜쮏얧왐햤햦혝 햩햦 혟햟햧햩 햡햣쮏쮏쥃썜햦쮏쫧쫨햣 혝햣햡햦.
